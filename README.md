# Love Site

Этот проект готов для публикации через GitHub Pages.

## Что уже подготовлено

- `index.html` открывает стартовую страницу сайта
- все ссылки между страницами относительные и подходят для GitHub Pages
- добавлен `.nojekyll`, чтобы GitHub Pages отдавал статические файлы без лишней обработки
- финальная страница использует `video.gif`, что надёжнее для телефона

## Как запустить через GitHub Pages

1. Создай новый репозиторий на GitHub.
2. Загрузи в него все файлы из этой папки.
3. Открой `Settings` -> `Pages`.
4. В `Build and deployment` выбери:
   `Source` -> `Deploy from a branch`
5. Выбери ветку:
   `main`
6. Папка:
   `/ (root)`
7. Нажми `Save`.

Через пару минут сайт появится по адресу вида:

`https://USERNAME.github.io/REPOSITORY-NAME/`
